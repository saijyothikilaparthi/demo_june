🌍 Environment Variables

What are Environment Variables?

Environment Variables are name-value pairs used by the Operating System and applications to store configuration information.

Example:
USERNAME = jyothi
HOME = /home/user

Why are they used?

  • Store system settings
  • Store user information
  • Locate software
  • Share configuration between programs

Common Variables

VariablePurpose
HOMEUser Home Folder
PATHCommand Search Path
USERCurrent User
PWDCurrent Directory
SHELLCurrent Shell

Create Variable


name="OpenAI"
