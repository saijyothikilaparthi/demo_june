🐧 Linux Commands

📖 Manual

man <command>

Opens the manual page of a command.

🧹 Clear Screen

clear

Clears the terminal screen.

👤 User & System

CommandPurpose
whoamiCurrent User
whoLogged-in Users
uname -aSystem Information

📂 Command Location

CommandPurpose
which lsExecutable Location
whereis lsExecutable + Manual

🖨 Printing & Time

echo "Hello"

echo -e "A\nB"

date

time ls

📁 Files & Folders

touch file.txt

ls -a

du -sh folder

rmdir folder

🔐 Permissions

chmod +x script.sh

chmod -x script.sh

Give or remove execute permission.

▶ Run Script

bash script.sh

./script.sh

🌐 Internet

curl https://example.com

curl -o page.html https://example.com

⚙ Process & PATH

kill <PID>

timedatectl

echo $PATH
Remember

Read Manual → Run Command → Try Options → Observe Output
