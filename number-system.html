🔢 Number System

What is a Number System?

A Number System is a method of representing numbers using a set of digits.

System Base Digits
Binary 2 0,1
Decimal 10 0-9
Octal 8 0-7
Hexadecimal 16 0-9,A-F

All number systems can be converted from one form to another.

Decimal → Binary

Repeated Division by 2

25 ÷ 2

25 → 12 R1

12 → 6 R0

6 → 3 R0

3 → 1 R1

1 → 0 R1

Answer = 11001₂

Binary → Decimal

Multiply each bit by powers of 2.


11001₂

=1×2⁴+1×2³+0×2²+0×2¹+1×2⁰

=16+8+0+0+1

=25₁₀

Decimal → Octal

Repeated Division by 8


25 ÷ 8

25 → 3 R1

3 → 0 R3

Answer = 31₈

Octal → Decimal

Multiply digits by powers of 8.


31₈

=3×8¹+1×8⁰

=24+1

=25₁₀

Decimal → Hexadecimal

Repeated Division by 16


25 ÷ 16

25 → 1 R9

1 → 0 R1

Answer = 19₁₆

Hexadecimal → Decimal

Multiply digits by powers of 16.


19₁₆

=1×16¹+9×16⁰

=16+9

=25₁₀
