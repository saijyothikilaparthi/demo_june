Git & GitHub

Local Git Commands & SSH Key Generation

What is Local Git?

Git tracks changes in your project on your local machine before sending them to GitHub.

Initialize a Repository

git init

Creates a new local Git repository.

Common Git Commands

git status
git add .
git commit -m "message"
git log
git diff

Configure Your Identity

git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"

Run only once on your machine.

Verify

git config --global --list

Generate SSH Key

ssh-keygen -t ed25519 -C "your-email@example.com"
Press Enter three times to accept the default settings.

Start SSH Agent

eval "$(ssh-agent -s)"

Starts the SSH agent.

Add Your Key

ssh-add ~/.ssh/id_ed25519

Adds your private key to the SSH agent.

Copy Public Key

cat ~/.ssh/id_ed25519.pub

Copy the displayed key.

Add Key to GitHub

GitHub → Profile → Settings → SSH and GPG Keys → New SSH Key

Paste the copied public key and save.

Trust GitHub

ssh -T git@github.com
When prompted, type: yes

Test SSH Connection

ssh -T git@github.com

Expected: 

Hi username!
You've successfully authenticated.

Clone Repository

git clone git@github.com:username/repository.git

Downloads the repository using SSH.

Verify Remote

git remote -v

Ensure the remote URL starts with git@github.com.

Debug SSH

ssh -vT git@github.com

Shows detailed SSH connection information.

Complete Git Workflow

git status
git add .
git commit -m "Describe your change"
git push origin main
Run these commands every time after making changes.
