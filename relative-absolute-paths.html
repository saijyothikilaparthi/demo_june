📂 Relative & Absolute Paths

What is a Path?

A Path is the location of a file or folder.

SymbolMeaning
/Root Folder
~Home Folder
.Current Folder
..Parent Folder

Absolute vs Relative Path

Absolute Relative
Starts with / Starts from Current Folder
Same everywhere Depends on current location
Full Address Short Address

Example Folder Structure

Desktop
└── Kutumbam
    ├── Peddamma
    │   ├── Ravi
    │   └── Sita
    └── Pinni
       ├── Kiran
       └── Geetha

Example

Current Folder: Ravi

Go To: Kiran

Relative Path 
cd ../../Pinni/Kiran
Absolute Path 
cd /c/Users/username/Desktop/Kutumbam/Pinni/Kiran

Useful Commands

pwd      → Print Current Directory

cd ~     → Home Directory

cd ..    → Parent Directory

cd -     → Previous Directory

cd       → Home Directory

ls       → List Files & Folders
Remember

✔ Starts with / → Absolute Path
✔ Starts with ., .. or folder name → Relative Path
